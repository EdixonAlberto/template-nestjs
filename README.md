# Template NestJS

Plantilla para crear proyectos con Node usando el framework NestJS

## Inicio

```sh
corepack enable
cp .env.template .env
pnpm install
```

## Comandos

```sh
# development
pnpm start

# watch mode
pnpm dev

# production mode
pnpm start:prod

# unit tests
pnpm test

# e2e tests
pnpm test:e2e

# test coverage
pnpm test:cov
```
